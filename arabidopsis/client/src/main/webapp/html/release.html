Arabidopsis Release Notes
 

The final Arabidopsis thaliana information network contains:

    Datasets used for integration

    • Gene
      • Athaliana_167_gene.gff3
    • Protein
      • Athaliana_167_protein.fa
      • Athaliana_167_cds.fa
      • Athaliana_167_synonym.txt
      • UniProt: taxonomy:3702 AND database:(type:tair)
    • Gene Ontology
      • GAF2Ondex.txt
      • gene_association.tair.gz
    • Gene Pathway
      • Plant Metabolic Network AraCyc 11.5
    • Gene Phenotype
      • Phenotypes
    • Gene PPI
      • Protein interactions
    • Gene Publication
      • Locus_Published_20130305.txt
    • Text Mining
      • to.obo
      • pubmed_result.xml for query: arabidopsis
    • References
      • Arabidopsis (TAIR) see Release notes for Arabidopsis
      • Plants (UniProt) - Information of well observed plant organisms excluding Arabidopsis
        UniProt: taxonomy:33090 NOT taxonomy:3702 AND reviewed:yes
    • Homology
      • Decypher
    • Ontologies
      • gene_ontology.1_2.obo
